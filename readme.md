## 📌 Sobre o Projeto

Projeto desenvolvido no programa Explorer da Rocketseat para calcular o índice de Massa Corpórea.

Assuntos importantes aplicados: 

  * ES6 Modules
  * Modal
  * Manipulação da DOM
